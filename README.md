# BookBazaar 📚

An online bookstore built with React + Node.js + MySQL.

## Live Demo
🌐 https://harsh9137.github.io/BOOK-BAZAAR/

## Tech Stack

**Frontend:** React, TypeScript, Vite, Tailwind CSS  
**Backend:** Node.js, Express, MySQL, Sequelize  

## Project Structure

```
BOOK-BAZAAR/
├── Bookbazaar/     # React frontend
└── backend/        # Node.js + Express API
```

## Local Setup

### Frontend
```bash
cd Bookbazaar
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
# Set up your MySQL database using backend/database/setup.sql
npm start
```

## Deploy Frontend to GitHub Pages
```bash
cd Bookbazaar
npm run deploy
```
